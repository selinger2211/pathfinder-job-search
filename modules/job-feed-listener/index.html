Connect Gmail Account

To scan Gmail for job opportunities, you'll need to provide a Gmail API access token.
Steps:
  1. Go to Google Account Permissions
  2. Create a project and enable Gmail API
  3. Generate an OAuth 2.0 token
  4. Paste the token below
Your token is stored locally in your browser — we never transmit it to external servers.
